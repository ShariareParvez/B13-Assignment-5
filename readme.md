## Questions

1. What is the difference between var, let, and const?

## Answers

var: Var is function or global scope and allows re-declaration and updates within the same scope.

let: Let is block scoped, allowing updates but not re-declaration within the same block.

const: Const also block-scoped that cannot be reassigned after their initial assignment.

 2. What is the spread operator (...)?

## Answers

The spread operator (...) is used to expand elements from arrays, strings, or objects. It helps make code cleaner.

## Questions

3. What is the difference between map(), filter(), and forEach()?

## Answers

map(): creates a new array by applying a specified function to each element of the original array.

filter(): creates a new array with elements that pass conditions.

forEach(): loops through the array.
